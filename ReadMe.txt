Created by Stefan Walther (http://www.qlikblog.at)

Detailed explanation with screenshots: http://www.qlikblog.at/1579/qliktip-39-displaying-html-content-minimalistic-htmltextbox-object-extension/

Instructions:
--------------------------------------------------
Enter a reference to the desired QlikView field containing the Html or insert your Html as string.

The following properties will be copied from the object:
- Font Size (if not overruled by CSS)
- Font Family (if not overruled by CSS)
- Font Style (if not overruled by CSS)

By default horizontal and vertical scrollbars will be added if necessary.
